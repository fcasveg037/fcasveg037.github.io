Esta página contiene enlaces a los diferentes ejercicios que voy a realizar para el módulo LMSGE del IES Guadalpeña.

Ejercicios HTML

  1. Prueba como queda visualmente todos los encabezados <H1> a <H6>
  2. Crea una página donde pruebes los elementos <p> con la etiqueta align (center, left, right) y <pre>
  3. Crea una página donde muestres la misma imagen con diferentes tamaños y utilizando la etiquetas border y align (al menos 4 veces)
  4. Crea un listado NO enumerado de grupos musicales, otro listado enumerado de webs que visites regurlármente y un listado de definiciones <dl> <dt> <dd>
  5. Crea una página donde utilices todos los distintos formatos de texto
  6. Crea un fichero con nombre EjercicioTablas.html y realiza las 3 partes
  7. Crea una página web que contenga un formulario
  8. Crea otra página web que contenga 2º formulario
  9. Realiza la práctica libre HTML

Ejercicios CSS

  1. Básico - Formulario
  2. Menús (Vertical y horizontal)
  3. Crea la composición 1-3-1
